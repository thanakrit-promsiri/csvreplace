พุทธบุตร! พระโพธิสัตว์นี้ย่อมสามารถแสดงพระธรรมอันเป็นสาวกยาน พระธรรมอันเป็นปัจเจกพุทธยาน พระธรรมอันเป็นโพธิสัตว์ยาน และพระธรรมอันเป็นภูมิแห่งพระตถาคตได้เป็นอย่างดี

一切行處，智隨行故，能隨眾生根性、欲解、所行有異、諸聚差別，

สรรพโคจร (ขอบเขตแห่งการดำเนินไปทั้งหลาย) ย่อมดำเนินไปตามญาณ เหตุนั้นย่อมสามารถคล้อยตามสัตว์ทั้งหลาย ตามอินทรีย์ อัธยาศัย ความประพฤติอันต่างกัน แลความแตกต่างแห่งบรรดากองธรรมทั้งหลาย

行處 (xíngchù): สถานที่แห่งการกระทำ ขอบเขตแห่งการดำเนินไป
隨行 (suíxíng): ดำเนินไปตาม, เป็นไปตาม



亦隨受生、煩惱、眠、縛、諸業習氣而為說法，令生信解，增益智慧，各於其乘而得解脫。 

亦 (yì) : อีกทั้ง / ยัง

隨 (suí) : ตาม / ขึ้นอยู่กับ

受生 (shòu shēng) : การไปเกิดในภพภูมิ / การรับกำเนิด

煩惱 (fán nǎo) : กิเลส / ความเร่าร้อน

眠 (mián) : ความหลับใหล (แห่งกิเลส) / อนุสัย (กิเลสที่นอนเนื่องอยู่ในสันดาน)

縛 (fù) : เครื่องผูกมัด / พันธนาการ (หมายถึงกิเลสที่ผูกมัดสรรพสัตว์ไว้)

諸業 (zhū yè) : กรรมทั้งหลาย

習氣 (xí qì) : อุปนิสัย / ความเคยชินที่สั่งสมมา / วาสนา

而 (ér) : แล้วจึง / เพื่อที่จะ

為說法 (wèi shuō fǎ) : แสดงธรรมแก่ / เทศนาแก่

令 (lìng) : ยังให้เกิด / ทำให้

生 (shēng) : เกิดขึ้น

信解 (xìn jiě) : ความเชื่อมั่นและเข้าใจ / ศรัทธาและความเข้าใจ

增益 (zēng yì) : เพิ่มพูน / เพิ่มพูนยิ่งขึ้น

智慧 (zhì huì) : ปัญญา

各 (gè) : แต่ละ / แตกต่างกันไป

於 (yú) : ใน

其乘 (qí chéng) : ยานของตน / ยานนั้น (หมายถึงยานที่แต่ละบุคคลมุ่งไป เช่น สาวกยาน ปัจเจกพุทธยาน โพธิสัตวยาน)

而得解脫 (ér dé jiě tuō) : แล้วจึงได้หลุดพ้น / บรรลุความหลุดพ้น
